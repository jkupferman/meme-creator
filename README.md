# meme creator

## check it out
meme creator is currently running at http://memeifier.com/

## dependencies
imagemagick
memcached

## getting started

     # install da magic
     brew install imagemagick

     # install da cache
     brew install memcached

     # install da gems
     bundle

     # run it
     shotgun app.rb

     # open it up in your browser
     open http://localhost:9393
